Game Description Please read carefully

Goal

The goal of this game is to find answers to several questions by searching the web using our provided search interface. In this contest you will need to use your best search skills, general knowledge, and quick thinking. We hope this contest is fun, and challenges you to use search creatively!


When searching for the answer you will be asked some additional questions on the quality of search results. Answers to these questions are a big part of the game.

Requirements

  • You should use web search to find answers to the provided questions (even if you know the answer)
  • You should only use provided search interface and do not try to issue this query to another search engine in a separate browser tab.
  • You can examine search results by clicking a link in search interface or right-clicking and opening it in a new tab. You should not copy url and enter this URL into browser address bar because we require all results to be viewed through our proxy.
  • You should provide trustful answers to the questions regarding search results quality
  • You are allowed to skip a question only if you put good effort in finding the answer and didn't succeed.
  • You shouldn't deliberately try to break the game website or use it in any other way other than for playing the game.

Payments

  • All requirements should be satisfied in order for a player to recieve his payment
  • We reserve the right to disqualify a player for violating one or more requirements listed above
  • A player can also be entitled for a bonus if all requirements are satisfied and answers to all questions are correct
