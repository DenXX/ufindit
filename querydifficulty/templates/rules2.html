Possible problems with a search document

Missing topic If information on some topic from the query is missing, check the corresponding checkbox. For example: if for query "Neiblemeister wife" a document doesn't talk about wife, you should check the corresponding checkbox.
Misinterpreted keywords If a document describes something that is a different interpretation of a term you used, check the corresponding checkbox. For example, if for query "Neiblemeister wife" a document talks about different person with the same name and his wife you should check Neiblemeister as misinterpreted.
Missing relation between topics If document have some information about topics from the query, but the connection between topics are missing, then you should check 2 or more checkboxes in this row for topics connection between which is missing. For example, if for query "Neiblemeister wife" a document has some information about the correct Neiblemeister and in some other contex about someone else wife, then the connection between topics is not captured and you should select checkboxes for the both topics. In the document shown above this is exactly the case, so we should check both checkboxes in the last row.
Other problem If you think that neither or above reasons apply but document doesn't answer the question and you didn't like it, please specify why.
