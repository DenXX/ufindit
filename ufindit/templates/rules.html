Game Description Please read carefully

Goal

The goal of this game is to find answers to several questions by searching the web using our provided search interface. In this contest you will need to use your best search skills, general knowledge, and quick thinking. We hope this contest is fun, and challenges you to use search creatively!


When searching for the answer you will be asked some additional questions on the quality of search results. Responses to these additional questions are an important part of the task. The quality of your responses will be used to decide the bonus amount, if any.

Requirements

  • You should use web search to find answers to the provided questions (even if you know the answer)
  • You should only use provided search interface and do not try to issue this query to another search engine in a separate browser tab. In order to receive credit for your work, the result page must be found and visited through our search interface only
  • Attempts to abuse the game or the provided search interface will be prosecuted with Amazon MTurk customer support.

Payments

  • All requirements should be satisfied in order for a player to recieve his payment.
  • Payments will be based on effort - sometimes the answer is very difficult to find, no matter how hard you try, and good faith effort is sufficient for base payment.
  • A player may be entitled to a bonus payment for correctly answering the game questions, and providing additional responses about search quality as requested.
